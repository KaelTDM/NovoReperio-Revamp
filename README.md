# Novo Reperio

Fresh project scaffold for the upcoming website redesign.

## Stack

- Astro
- Tailwind CSS v4

## Structure

- `src/pages` route files
- `src/layouts` shared page shells
- `src/components` reusable UI
- `src/data` structured content
- `src/styles` global styles
- `public` static assets

## Commands

```bash
npm run dev
npm run build
npm run preview
```
